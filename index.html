Obstacle Avoiding Robot

Description: An autonomous robot that detects and avoids obstacles using ultrasonic sensors. It navigates through unknown environments without human intervention.

Technologies:

  • Microcontroller: Arduino, ATmega32
  • Sensors: Ultrasonic Sensor (HC-SR04)
  • Motor Driver: L298N
  • Programming: C, C++

Features:

  • Real-time obstacle detection.
  • Smooth navigation using PWM motor control.
  • Low-cost and efficient design.

Challenges:

  • Calibrating the ultrasonic sensor for accurate distance measurement.
  • Ensuring smooth motor control and obstacle avoidance.
